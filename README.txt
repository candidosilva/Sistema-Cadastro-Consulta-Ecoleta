PROJETO NLW - NEXT LEVEL WEEK

Projeto de um site de cadastro de empresas de reciclagem.
Trata-se de um projeto de estudo feito durante o NLW da Rocketseat.

Tecnologias utilizadas:
	HTML
	CSS
	JAVASCRIPT
	NODEJS
	BANCO DE DADOS SQLITE3
	NUNJUCKS-TEMPLATE
	API DE LOCALIDADES DO IBGE
	
Nesse projeto podemos cadastrar novas empresas e registrá-las por tipo de serviço de reciclagem.

Também podemos Pesquisar as empresas cadastradas por cidade utilizando a API de localidades do IBGE.
